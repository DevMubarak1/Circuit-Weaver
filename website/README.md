# Circuit Weaver — Website

Marketing website for [Circuit Weaver](https://github.com/DevMubarak1/Circuit-Weaver), built with Next.js 15, Tailwind CSS v4, and Framer Motion.

## Features

- Interactive logic gate demo on the homepage (AND, OR, XOR — click to toggle inputs)
- Per-character and word-level text animations with scroll-triggered reveals
- Bento grid feature cards, tab-based gate explorer, chapter progression cards
- Embedded game player at `/play` with fullscreen support
- Fully responsive across mobile, tablet, and desktop
- Space Grotesk + JetBrains Mono typography
- Midnight Architect dark theme with glassmorphic panels

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Stack

- **Next.js 15** — App Router, static generation
- **Tailwind CSS v4** — utility-first styling
- **Framer Motion** — scroll animations, spring physics, layout animations
- **TypeScript** — strict type checking
